# فراخوان‌های سیستمی پشتیبانی‌شده

فراخوان‌های سیستمی زیر در حال حاضر پیاده‌سازی شده‌اند. شماره‌ها با جدول فراخوان سیستمی لینوکس x86_64 برای سازگاری مطابقت دارند.

| شماره | نام | توضیح |
| --- | --- | --- |
| 0 | `read` | خواندن از fd |
| 1 | `write` | نوشتن در fd |
| 2 | `open` | باز کردن فایل |
| 3 | `close` | بستن fd |
| 12 | `brk` | تنظیم انتهای heap |
| 22 | `pipe` | ایجاد pipe |
| 39 | `getpid` | گرفتن شناسه‌ی فرایند |
| 57 | `fork` | فورک فرایند |
| 59 | `execve` | اجرای باینری ELF |
| 60 | `exit` | خروج از فرایند |
| 61 | `wait4` | انتظار برای فرزند |
| 62 | `kill` | ارسال سیگنال |
| 110 | `getppid` | گرفتن PID والد |
| 162 | `nanosleep` | خواب |

برای قرارداد فراخوانی رجیستری به [ABI فراخوان سیستمی](./abi) مراجعه کنید.
