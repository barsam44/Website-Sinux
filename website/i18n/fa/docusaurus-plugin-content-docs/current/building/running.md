# اجرای Sinux

پس از [ساخت](./build) ISO، می‌توانید Sinux را در QEMU بوت کنید.

## حالت BIOS

```bash
make run-bios
```

## حالت UEFI

```bash
make run-uefi
```

## بدون gösterge / سریال

با هدایت خروجی به ترمینال اجرا کنید (مناسب برای SSH یا لاگ‌گیری):

```bash
make run-serial
```

## دیباگ

با فعال بودن لاگ割込 QEMU اجرا کنید:

```bash
make run-debug
```

:::tip سخت‌افزار واقعی
Sinux می‌تواند روی سخت‌افزار واقعی x86_64 هم بوت شود. ISO را روی یک درایو USB بنویسید و مطمئن شوید
**Secure Boot** در تنظیمات فیروزار خاموش است.
:::

پس از بوت، وارد شل داخلی با دستورات سیستم فایل می‌شوید. برای درک آنچه در زیرسطح اجرا می‌شود،
[مستندات معماری](/docs/architecture/overview) را بررسی کنید.
