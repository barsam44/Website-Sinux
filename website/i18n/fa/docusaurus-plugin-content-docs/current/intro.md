# Sinux چیست؟

**Sinux** یک هسته‌ی سیستم‌عامل کمینه‌ی ۶۴ بیتی شبه‌یونیکس است که از صفر با **C** و **اسمبلی x86_64** نوشته شده است. این هسته از طریق GRUB Multiboot2 بوت می‌شود، از **UEFI و BIOS** پشتیبانی می‌کند و روی سخت‌افزار واقعی یا QEMU اجرا می‌شود.

:::info این لینوکس نیست
Sinux برنامه‌های لینوکس را اجرا **نمی‌کند**. این یک هسته‌ی مستقل و آموزشی است که برای درک نحوه‌ی کار سیستم‌عامل‌های واقعی در پایین‌ترین سطح ساخته شده است.
:::

## امکانات

Sinux موارد زیر را خودش پیاده‌سازی می‌کند:

- 🧠 **مدیر حافظه‌ی فیزیکی و مجازی** (PMM + VMM)
- 📁 **VFS** (سیستم فایل مجازی) با پشتیبانی از `ramfs` و `ext2`
- 💾 **درایو دیسک ATA PIO** برای ذخیره‌سازی پایدار
- ⌨️ درایوهای **کیبورد PS/2** و **حالت متنی VGA**
- 🔌 خروجی دیباگ **سریال (COM1)**
- ⚙️ جداول سطح‌پایین: **IDT، GDT، TSS، PIC، PIT**
- 🧩 **رابط فراخوان سیستمی** (سازگار با ABI لینوکس x86_64)
- 🔁 **زمان‌بند فرایند** (round-robin)
- 📦 **بارگذار ELF64** برای برنامه‌های فضای کاربر
- 🛡️ اجرای **Ring 3** (حالت کاربر)
- 🔗 بدوی‌های **pipe و سیگنال**
- 🐚 **شل** داخلی با دستورات سیستم فایل
